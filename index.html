HTML

HTML es la sigla de HyperText Markup Language, el cual nos ayuda a darle la estructura a nuestro sitio web por medio de etiquetas, lo mejor de este lenguaje es su fácil interpretación.

Aprender HTML es fundamental para crear sitios web(mencionado anteriormente), el cual podremos utilizarlo en conjunto de otras tecnologías, como CSS y/o Javascript


Elementos y Estructuras

Anatomía de HTML

HTML trabaja con elementos, estos elementos contienen la estructura y contenido de un sitio web:

La imagen de arriba contiene los elementos de párrafo, en este caso nos ayuda a colocar párrafo o texto dentro de nuestro sitio, la gran mayoría de las etiquetas o tag comienzan con un tag de apertura y un tag de cierre, el contenido que muestra en el archivo o pagina web es de "Mi gato es muy gruñon"

Para que quede un poco más claro resumiremos los mencionado anteriormente:

  • Elemento HTML: Es un grupo que contiene contenido, en este caso las etiquetas de apertura y cierre junto al contenido que almacenan para desplegarlo en el sitio web.

  • Etiqueta o tag HTML:Es el nombre de elemento que se identifica por las etiquetas de apertura y cierre.

  • Contenido: es la información que se mostrará en un sitio web, ya sea texto, vínculo, imagen, video, etc.
